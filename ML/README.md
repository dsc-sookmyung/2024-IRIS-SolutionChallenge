# Missing You-ML

## Installation
### - environment
> Ensure that Python 3.9+ is installed on your system. You can check this by using: `python --version`.  

```bash
conda create -n MissingYouML python=3.9
conda activate MissingYouML
```
### dependencies
```bash
pip install -r requirements.txt
```

## Run
```bash
uvicorn main:app
```
